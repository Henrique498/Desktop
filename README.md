# 📚 Buscador de Livros — Google Books API

Aplicação **desktop** desenvolvida em Python que permite pesquisar livros em tempo real utilizando a **API do Google Books**. O projeto conta com uma interface gráfica construída com **PyQt**, estilizada com QSS.

---

## ✨ Funcionalidades

- 🔍 Pesquisa de livros por título, autor ou palavra-chave
- 📖 Exibição de informações como título, autor, descrição e capa
- 🎨 Interface gráfica estilizada com QSS (Qt Style Sheets)
- 🌐 Integração com a Google Books API

## 🚀 Como executar

```bash

git clone https://github.com/Henrique498/Desktop.git

# Acesse a pasta
cd Desktop

# Instale as dependências
pip install PyQt5 requests

# Execute a aplicação
python api.py
```
